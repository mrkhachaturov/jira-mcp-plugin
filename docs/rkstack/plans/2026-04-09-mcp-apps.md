# MCP Apps Implementation Plan

> **For agentic workers:** Use subagent-driven-development (recommended) or executing-plans skill to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Spec:** `docs/rkstack/specs/2026-04-08-mcp-apps-design.md`

**Goal:** Add MCP Apps interactive UI support so Jira issues render as rich, actionable cards inside Claude, ChatGPT, and other MCP Apps-compatible clients.

**Architecture:** Java server-side changes (ResourceRegistry, JsonRpcHandler, McpTool interface) provide `resources/read` for `ui://` resources and add `_meta.ui`/`annotations`/`structuredContent` to tool responses. A separate React/Vite project (`mcp-app/`) builds a single-file HTML widget bundled into the JAR classpath. The widget communicates with the server via the MCP Apps postMessage bridge.

**Tech Stack:** Java 17 (JAX-RS, javax.\*), React 19, TypeScript, Vite + vite-plugin-singlefile, @modelcontextprotocol/ext-apps

**Commands:**
- Build: `just build` (or `atlas-package -DskipTests`)
- Test: `just e2e` (or `atlas-mvn test -Dtest=McpEndpointE2ETest -DfailIfNoTests=false`)
- Deploy: `just deploy`
- Widget dev: `just dev-app`

**Known issues from dual-review (to resolve during execution):**

1. **CI workflow** — `.github/workflows/build.yml` uses `mvn package` directly. Add Node.js setup + `npm ci && npm run build` step before `mvn package`, or use Maven exec plugin.
2. **Lockfile** — `npm install` in Task 9 generates `package-lock.json`. Commit it so `npm ci` works in CI/`just build-app`.
3. **Field union** — UI-linked tools must union caller fields with UI-required fields before calling Jira. Add field augmentation logic in each tool's `execute()` or in a shared helper.
4. **GetProjectIssuesTool is broken** — it ignores `project_key`, `limit`, `start_at` params (line 50 calls bare `/rest/api/2/search`). Fix it to build JQL `project = KEY` with proper params before marking it UI-capable.
5. **Base URL fallback** — `ResourceRegistry` and `buildStructuredContent()` must inject `ApplicationProperties` for detected Jira base URL when no override is configured. Don't return empty string.
6. **structuredContent completeness** — use `total` from Jira search responses (not `issues.size()`), include optional fields (`labels`, `components`, `fix_versions`), populate `currentUser.displayName`.
7. **Transition enrichment** — `GetTransitionsTool` returns a JSON array (not `{transitions:[...]}`). Widget must parse array directly. For `hasRequiredFields`, call Jira's `?expand=transitions.fields` and check for required fields server-side. Add this as a prerequisite sub-task.
8. **E2E test ordering** — move MCP Apps tests after build pipeline tasks so widget HTML is in classpath. Make assertions unconditional for full-build scenarios.
9. **Username pass-through** — `JsonRpcHandler.handle()` signature needs `username` added. Update `McpResource.java` call site to pass `username` alongside `userKey`. The plan's inline comment must become actual code.

```java
package com.atlassian.mcp.plugin;

import com.atlassian.mcp.plugin.config.McpPluginConfig;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.databind.node.ArrayNode;
import com.fasterxml.jackson.databind.node.ObjectNode;
import javax.inject.Inject;
import javax.inject.Named;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import java.io.InputStream;
import java.nio.charset.StandardCharsets;
import java.security.MessageDigest;
import java.util.LinkedHashMap;
import java.util.Map;

@Named
public class ResourceRegistry {

    private static final Logger log = LoggerFactory.getLogger(ResourceRegistry.class);
    private static final String HTML_CLASSPATH = "/mcp-app/issue-card.html";
    private static final String MIME_TYPE = "text/html;profile=mcp-app";

    private final McpPluginConfig config;
    private final ObjectMapper mapper = new ObjectMapper();

    private String html;
    private String resourceUri;
    private Map<String, Object> resourceMeta;

    @Inject
    public ResourceRegistry(McpPluginConfig config) {
        this.config = config;
        loadWidget();
    }

    private void loadWidget() {
        try (InputStream is = getClass().getResourceAsStream(HTML_CLASSPATH)) {
            if (is == null) {
                log.warn("[MCP-APPS] Widget HTML not found at classpath:{} — MCP Apps disabled", HTML_CLASSPATH);
                return;
            }
            html = new String(is.readAllBytes(), StandardCharsets.UTF_8);
            String hash = sha256(html).substring(0, 12);
            resourceUri = "ui://jira/issue-card@" + hash;
            resourceMeta = buildMeta();
            log.info("[MCP-APPS] Widget loaded: {} ({} bytes)", resourceUri, html.length());
        } catch (Exception e) {
            log.warn("[MCP-APPS] Failed to load widget HTML: {}", e.getMessage());
        }
    }

    /** Whether the widget is available. */
    public boolean isAvailable() {
        return html != null && resourceUri != null;
    }

    /** The ui:// URI for this resource. */
    public String getResourceUri() {
        return resourceUri;
    }

    /** Build the resources/list result. */
    public String buildResourcesList() {
        if (!isAvailable()) {
            ObjectNode result = mapper.createObjectNode();
            result.set("resources", mapper.createArrayNode());
            return serialize(result);
        }

        ObjectNode result = mapper.createObjectNode();
        ArrayNode resources = mapper.createArrayNode();
        ObjectNode resource = mapper.createObjectNode();
        resource.put("uri", resourceUri);
        resource.put("name", "jira-issue-card");
        resource.put("title", "Jira Issue Card");
        resource.put("description", "Interactive Jira issue viewer with status transitions and comments");
        resource.put("mimeType", MIME_TYPE);
        resource.set("_meta", mapper.valueToTree(resourceMeta));
        resources.add(resource);
        result.set("resources", resources);
        return serialize(result);
    }

    /** Build the resources/read result for a given URI. Returns null if URI not found. */
    public String buildResourceRead(String uri) {
        if (!isAvailable() || !resourceUri.equals(uri)) {
            return null;
        }

        ObjectNode result = mapper.createObjectNode();
        ArrayNode contents = mapper.createArrayNode();
        ObjectNode content = mapper.createObjectNode();
        content.put("uri", resourceUri);
        content.put("mimeType", MIME_TYPE);
        content.put("text", html);
        content.set("_meta", mapper.valueToTree(resourceMeta));
        contents.add(content);
        result.set("contents", contents);
        return serialize(result);
    }

    private Map<String, Object> buildMeta() {
        String baseUrl = getBaseUrl();
        Map<String, Object> meta = new LinkedHashMap<>();

        // MCP Apps standard (Claude, VS Code, Goose)
        Map<String, Object> ui = new LinkedHashMap<>();
        ui.put("prefersBorder", true);
        Map<String, Object> csp = new LinkedHashMap<>();
        csp.put("connectDomains", java.util.List.of());
        csp.put("resourceDomains", java.util.List.of());
        ui.put("csp", csp);
        meta.put("ui", ui);

        // OpenAI / ChatGPT compatibility
        meta.put("openai/widgetDescription", "Interactive Jira issue viewer with status transitions and comments");
        meta.put("openai/widgetPrefersBorder", true);
        Map<String, Object> openaiCsp = new LinkedHashMap<>();
        openaiCsp.put("connect_domains", java.util.List.of());
        openaiCsp.put("resource_domains", java.util.List.of());
        meta.put("openai/widgetCSP", openaiCsp);
        meta.put("openai/widgetDomain", baseUrl);

        return meta;
    }

    private String getBaseUrl() {
        String override = config.getJiraBaseUrlOverride();
        if (override != null && !override.isEmpty()) return override;
        return "";
    }

    private String serialize(Object obj) {
        try {
            return mapper.writeValueAsString(obj);
        } catch (Exception e) {
            return "{}";
        }
    }

    private static String sha256(String input) {
        try {
            MessageDigest md = MessageDigest.getInstance("SHA-256");
            byte[] hash = md.digest(input.getBytes(StandardCharsets.UTF_8));
            StringBuilder hex = new StringBuilder();
            for (byte b : hash) hex.append(String.format("%02x", b));
            return hex.toString();
        } catch (Exception e) {
            return "000000000000";
        }
    }
}
```

```java
private String handleResourcesRead(JsonNode id, JsonNode params) {

    if (uri == null || uri.isEmpty()) {
        return errorResponse(id, -32602, "Missing 'uri' in params");

    try {
        return successResponse(id, mapper.readTree(result));
    } catch (Exception e) {
        return errorResponse(id, -32603, "Failed to read resource");
    }
}
```

```java
private ObjectNode buildStructuredContent(String toolName, JsonNode jiraResponse, String username) {
    ObjectNode structured = mapper.createObjectNode();

    // currentUser — uses username (not userKey) for Jira assignee compatibility
    ObjectNode currentUser = mapper.createObjectNode();
    currentUser.put("name", username != null ? username : "");
    structured.set("currentUser", currentUser);

    // baseUrl
    String baseUrl = config.getJiraBaseUrlOverride();
    if (baseUrl == null || baseUrl.isEmpty()) {
        baseUrl = "";
    }
    structured.put("baseUrl", baseUrl);

    // Normalize issues into common array
    ArrayNode issues = mapper.createArrayNode();

    if ("get_issue".equals(toolName)) {
        // Single issue response — wrap in array
        ObjectNode normalized = normalizeIssue(jiraResponse);
        if (normalized != null) issues.add(normalized);
    } else {
        // Search/list response — extract issues array
        JsonNode issuesNode = jiraResponse.path("issues");
        if (issuesNode.isArray()) {
            for (JsonNode issue : issuesNode) {
                ObjectNode normalized = normalizeIssue(issue);
                if (normalized != null) issues.add(normalized);
            }
        }
    }

    structured.set("issues", issues);
    structured.put("totalCount", issues.size());

    return structured;
}

private ObjectNode normalizeIssue(JsonNode issue) {
    if (issue == null || issue.isMissingNode()) return null;

    ObjectNode n = mapper.createObjectNode();
    n.put("key", issue.path("key").asText(""));

    JsonNode fields = issue.has("fields") ? issue.get("fields") : issue;

    n.put("summary", fields.path("summary").asText(""));

    // Status
    ObjectNode status = mapper.createObjectNode();
    JsonNode statusNode = fields.path("status");
    status.put("name", statusNode.path("name").asText(""));
    status.put("category", statusNode.path("statusCategory").path("key").asText(""));
    n.set("status", status);

    // Priority
    ObjectNode priority = mapper.createObjectNode();
    priority.put("name", fields.path("priority").path("name").asText(""));
    n.set("priority", priority);

    // Issue type
    ObjectNode issueType = mapper.createObjectNode();
    JsonNode typeNode = fields.has("issuetype") ? fields.get("issuetype") : fields.path("issue_type");
    issueType.put("name", typeNode.path("name").asText(""));
    n.set("issue_type", issueType);

    // Assignee
    JsonNode assigneeNode = fields.path("assignee");
    if (assigneeNode.isNull() || assigneeNode.isMissingNode()) {
        n.putNull("assignee");
    } else {
        ObjectNode assignee = mapper.createObjectNode();
        assignee.put("name", assigneeNode.path("name").asText(""));
        assignee.put("displayName", assigneeNode.path("displayName").asText(""));
        n.set("assignee", assignee);
    }

    // Reporter
    JsonNode reporterNode = fields.path("reporter");
    if (reporterNode.isNull() || reporterNode.isMissingNode()) {
        n.putNull("reporter");
    } else {
        ObjectNode reporter = mapper.createObjectNode();
        reporter.put("name", reporterNode.path("name").asText(""));
        reporter.put("displayName", reporterNode.path("displayName").asText(""));
        n.set("reporter", reporter);
    }

    // Description
    n.put("description", fields.path("description").asText(""));

    // Comments
    ArrayNode comments = mapper.createArrayNode();
    JsonNode commentContainer = fields.path("comment");
    JsonNode commentList = commentContainer.has("comments") ? commentContainer.get("comments") : commentContainer;
    if (commentList.isArray()) {
        int count = 0;
        for (JsonNode c : commentList) {
            if (count++ >= 5) break;
            ObjectNode comment = mapper.createObjectNode();
            comment.put("author", c.path("author").path("displayName").asText(""));
            comment.put("body", c.path("body").asText(""));
            comment.put("created", c.path("created").asText(""));
            comments.add(comment);
        }
    }
    n.set("comments", comments);

    // Timestamps
    n.put("created", fields.path("created").asText(""));
    n.put("updated", fields.path("updated").asText(""));

    return n;
}
```

```tsx
  const { app, isConnected } = useApp({

        setData(null)
        setError(null)

      app.ontoolresult = (params) => {
        setLoading(false)
        if (params.isError) {

          return
        }
        if (params.structuredContent) {
          setData(params.structuredContent as StructuredContent)
        }
      }
    },
```

```tsx
import { useState, useEffect } from 'react'
import type { App as McpApp } from '@modelcontextprotocol/ext-apps'

interface Transition {
  id: string
  name: string
  hasRequiredFields?: boolean
}

interface Props {
  app: McpApp
  issueKey: string
  onTransitioned: () => void
}

export function TransitionPicker({ app, issueKey, onTransitioned }: Props) {
  const [transitions, setTransitions] = useState<Transition[]>([])
  const [loading, setLoading] = useState(false)
  const [open, setOpen] = useState(false)

  useEffect(() => {
    if (!open) return
    setLoading(true)
    app.callServerTool({
      name: 'get_transitions',
      arguments: { issue_key: issueKey },
    }).then(result => {
      if (result?.structuredContent) {
        // structuredContent may not exist for get_transitions;
        // fall back to parsing text content
      }
      // Parse transitions from text response
      // v1: show all transitions. Server-side hasRequiredFields enrichment
      // is deferred to v2. If a transition requires screen fields, the
      // transition_issue call will fail and the widget shows an error.
      try {
        const text = (result as any)?.content?.[0]?.text ?? '{}'
        const data = JSON.parse(text)
        const raw = data.transitions ?? []
        setTransitions(raw.map((t: any) => ({
          id: String(t.id),
          name: t.name,
        })))
      } catch {
        setTransitions([])
      }
    }).finally(() => setLoading(false))
  }, [open, app, issueKey])

  const handleTransition = async (transitionId: string) => {
    await app.callServerTool({
      name: 'transition_issue',
      arguments: { issue_key: issueKey, transition_id: transitionId },
    })
    setOpen(false)
    onTransitioned()
  }

  if (!open) {
    return <button onClick={() => setOpen(true)}>Transition</button>
  }

  return (
    <div style={{ display: 'flex', gap: '4px', flexWrap: 'wrap' }}>
      {loading ? <span>Loading...</span> : transitions.map(t => (
        <button key={t.id} onClick={() => handleTransition(t.id)}
                className="primary" style={{ fontSize: '11px' }}>
          {t.name}
        </button>
      ))}
      <button onClick={() => setOpen(false)} style={{ fontSize: '11px' }}>Cancel</button>
    </div>
  )
}
```

```tsx
  issueKey: string
  onCommented: () => void
}

export function CommentForm({ app, issueKey, onCommented }: Props) {
  const [open, setOpen] = useState(false)
  const [body, setBody] = useState('')
  const [submitting, setSubmitting] = useState(false)

  const handleSubmit = async () => {
    if (!body.trim()) return
    setSubmitting(true)
    await app.callServerTool({
      name: 'add_comment',
      arguments: { issue_key: issueKey, body },
    })
    setBody('')
    setOpen(false)
    setSubmitting(false)
    onCommented()
  }

  if (!open) {
    return <button onClick={() => setOpen(true)}>Comment</button>
  }

  return (
    <div style={{ display: 'flex', gap: '4px', flex: 1 }}>
      <input
        value={body}
        onChange={e => setBody(e.target.value)}
        placeholder="Add a comment..."
        disabled={submitting}
        style={{
          flex: 1,
          padding: '4px 8px',
          border: '1px solid var(--border)',
          borderRadius: '4px',
          fontSize: '13px',
        }}
        onKeyDown={e => e.key === 'Enter' && handleSubmit()}
      />
      <button onClick={handleSubmit} disabled={submitting || !body.trim()} className="primary">
        {submitting ? '...' : 'Send'}
      </button>
      <button onClick={() => { setOpen(false); setBody('') }}>Cancel</button>
    </div>
  )
}
```

```tsx
      arguments: {
        issue_key: issue.key,
        fields: JSON.stringify({ assignee: currentUser.name }),

    onRefresh()
  }

  const isAssignedToMe = issue.assignee?.name === currentUser.name

  return (
    <div style={{

      flexWrap: 'wrap',
    }}>
      <TransitionPicker app={app} issueKey={issue.key} onTransitioned={onRefresh} />
      <CommentForm app={app} issueKey={issue.key} onCommented={onRefresh} />
      {!isAssignedToMe && (
        <button onClick={handleAssign}>Assign to me</button>
      )}
    </div>
  )
}
```
